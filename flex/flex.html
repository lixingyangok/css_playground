这是一个span 
        ● flex 容器独立成行，即使右侧有剩余空间也不容后续元素侵占（和 block 元素相同）
        ● 见上方， 元素没能挤上来
        ● flex 容器会使子元素收缩，且横向排列，即使子元素是 block & width: 9999px
    

@scope{
    .container{
        display: flex;
        width: 30%;
        & > *{
            /* width: 50px; */
            width: 9999px; /*  */
        }
    }
    .span{
        border: solid 1px
    }
}
